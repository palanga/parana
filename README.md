zio-event-sourcing
==================

An event sourcing library on top of ZIO
---------------------------------------

* Scala 2.13.4
* ZIO 1.0.3

Installation
------------

Add this to your `build.sbt` file
```sbt
resolvers += Resolver.bintrayRepo("palanga", "maven")
libraryDependencies += "dev.palanga" %% "zio-event-sourcing" % "version"
```
